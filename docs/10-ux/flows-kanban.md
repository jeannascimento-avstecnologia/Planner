# Fluxos Kanban (MVP)

1. Criar projeto (board) na lista `/boards`.
2. Abrir projeto → colunas + cards.
3. Criar card rapido (titulo, prioridade, prazo opcional).
4. Clicar card → drawer (editar tudo + tags + responsavel).
5. Filtrar por prioridade / marcador / responsavel; toggle agrupar por responsavel.
6. Compartilhar → convite por email + link `/invite?token=`.
