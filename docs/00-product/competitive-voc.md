# Competitive Intelligence / Voice of Customer (2026)

## Janela de mercado
- Trello: backlash do redesign 2025 (Trustpilot ~2.7/5); Atlassian declarou pivot para produtividade pessoal e parou de aceitar features de PM de time.
- Microsoft Planner (update jan-fev/2026): aposentou Whiteboard, feed iCal (sem substituto), Loop e Viva Goals; trocou comentarios no card por "task chat" (revolta); removeu Calendar grid do Premium.
- Conclusao: os dois lideres estao expulsando o B2B de times -> nossa cunha.

## Dores ranqueadas (com nossa jogada)
1. Performance/lentidao = dor nº1 ("usuarios nao perdoam lentidao") -> optimistic UI sub-100ms, fractional indexing.
2. Offline que nao funciona -> PWA offline-first + sync.
3. Relatorios fracos / sem rollup cross-projeto (Asana so por projeto) -> dashboard event-sourced (cross-board no fast-follow).
4. Notification overload -> digest/granular/DND por padrao.
5. Trello sem subtarefas e sem dependencias -> 1a classe no modelo de dados.
6. Mobile de 2a classe -> mobile-first real.
7. Whiteboards (Miro/FigJam) travam -> budget de performance (virtualizacao).
8. Automacao com quota punitiva (Butler) -> motor de regras sem teto (fast-follow).

## Principios anti-erro (licoes Planner/Trello)
- Comentarios SEMPRE no card (nunca chat).
- Calendario + iCal mantidos em todos os planos.
- Nunca remover/forcar mudanca em feature que o usuario depende.
- Performance e feature, nao detalhe.
